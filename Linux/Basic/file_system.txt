1. pwd - print working directory

2. cd [name] - change directory (go back: cd ..)

3. ls [name or leave blanck for list of current directory] - list of items in directory (show everything with hidden: ls -la)

4. mkdir/rmdir [name] - create or remove directory

5. > [name] / touch [name] - create new file (create file with short text in it: echo "short text" > file.txt; append to file: echo "text" >> file.txt)

6. cp [name] [new place] - copy files or directory and past them in other places

7. rm [name] - remove file 

8. mv/mvdir [name] [new place] - move file or directory to other place

9. locate [name] - lacet specific file or directory

10. updatedb - updates file system database

11. ranger - mini file manager (if not installed: sudo apt-get install ranger)

12. nano [name] - opens nano editor to edit specific file

13. vi [name] - opens vim editor to edit specific file

14. cat [name] - shows file content
